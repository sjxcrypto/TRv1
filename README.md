# TRv1

**High-performance sovereign L1 blockchain built on SVM**

TRv1 is a next-generation Layer 1 blockchain forked from the [Agave](https://github.com/anza-xyz/agave) Solana validator client. It preserves the raw throughput of the Solana Virtual Machine while replacing the consensus layer, economic model, and hardware requirements to create a more accessible and sustainable chain.

## Hardware Requirements

| Tier | RAM | CPU | Storage | Use Case |
|------|-----|-----|---------|----------|
| **Lite** | 32 GB | 8 cores | 500 GB NVMe | Standby validator, RPC node |
| **Standard** | 64 GB | 16 cores | 1 TB NVMe | Active validator |
| **Performance** | 128 GB | 32 cores | 2 TB NVMe | High-throughput validator, archival |

> **Note:** These are dramatically lower than Solana's recommended 512GB RAM / 24+ cores, thanks to TRv1's tiered storage architecture and BFT consensus reducing memory pressure.

## Building from Source


## Architecture

TRv1 retains the Solana Virtual Machine (SVM) execution layer â€” including Sealevel parallel scheduling, the BPF/SBF program runtime, and the accounts database â€” while replacing the consensus and economic layers:

| Layer | Solana/Agave | TRv1 |
|-------|-------------|------|
| **Consensus** | Proof-of-History + Tower BFT | Tendermint-style BFT |
| **Finality** | ~12.8s (probabilistic) | Instant (deterministic) |
| **Execution** | SVM / Sealevel | SVM / Sealevel (same) |
| **Min RAM** | 512 GB | 32 GB |
| **Staking Yield** | Variable (inflation-based) | Flat 5% per-coin |
| **Slashing** | Validators + Delegators | Validators only |
| **Fee Model** | Priority fees | EIP-1559 + 4-way split |
| **Dev Incentives** | Grants only | Automatic fee sharing |
